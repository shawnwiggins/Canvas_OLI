R Environment wtih OLI Data

With this environment, you can work through the coding exercises in the Open Learning Initiative's Probability & Statstics workbook.

Click inside the coding evironment to load all the data sets from OLI and wait for the yellow circle to turn green to know the evionment is ready to go. For example, the first activity works with the dataset friends. If you type friends in the left-side scipt panel and press Run, then you'll be able to access that data.
